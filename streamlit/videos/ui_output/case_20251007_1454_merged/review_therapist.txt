1) 主要所見（要点・左右差・大振幅）
- 歩幅に明確な左右差あり：step_length_r 7.191512 cm、step_length_l 3.472552 cm（px: R 35.958 / L 17.363）
- 歩幅非対称性が大きい：step_asymmetry 0.697475（高い非対称性を示唆）
- 速度は著しく低下：speed_m_s 0.044、speed_px_s 22.218（極めて低速な歩行）
- 骨盤の横揺れが大きい：pelvis_sway 107.592894（左右方向の体幹/骨盤変動が顕著）
- 肩の左右差は中等度：shoulder_diff_mean 4.764156、ばらつき shoulder_diff_std 4.029963
- かかと傾斜は左で平均・変動ともやや大きい：heel_tilt_r_mean 0.0822 / l_mean 0.114457、std R 0.065794 / L 0.117235
- 膝の立位方向指標は左右ほぼ同値：knee_varus_valgus_r_mean 176.114199、l_mean 176.340588（大きな左右差なし）
- 足部回内指標は左右ほぼ同等：ankle_pronation_r_mean 159.40486、l_mean 158.979043

2) 前額面（XY）: 股関節 内転/外転、膝 内反/外反、足関節 回内/回外
- 股関節（前額面）は直接指標なしだが、左歩幅短縮と骨盤横揺れ(pelvis_sway 107.59)から立脚側の代償的な股関節内転/外転の偏りが存在する可能性
- 膝（内反/外反）：knee_varus_valgus R 176.114199° / L 176.340588° — 左右で大きな差はなく、わずかな傾向があっても小幅
- 足関節（回内/回外）：ankle_pronation R 159.40486 / L 158.979043 — 回内（pronation）的な指標を示しており、左右差は小さい
- 骨盤横揺れが大きいため、前額面での荷重移動・側方制御の問題が示唆される

3) 矢状面（XZ）: 股関節 屈曲/伸展、膝 屈曲/伸展、足関節 背屈/底屈
- 歩行速度と著しい歩幅短縮（特に左）から、立脚終期での股関節伸展不足（推進力低下）が疑われる（直接の角度データは無し）
- 膝の伸展/屈曲：直接データなしだが、低速度・短歩幅は蹴り出し（立脚終期の伸展・伸長）不足や立脚期での膝屈曲保持を示唆
- 足関節（背屈/底屈）：heel_tilt_mean R 0.0822 / L 0.114457、std 左側が大きく、左で接地位相や底屈のタイミング変動がある可能性
- 全体として矢状面での前後推進が低いため、底屈（プッシュオフ）不足が機能的要因として想定される

4) 水平面（YZ, 近似）: 股関節・膝 内旋/外旋（可能な範囲で）
- 水平面回旋は直接計測値がないため確定不能だが、左右歩幅差と骨盤回旋の代償（pelvis_sway 107.59）より回旋運動の偏りが存在する可能性
- 肩差・骨盤横揺れがあるため、体幹の回旋/逆回旋の調整不足が示唆される（左右での過度な回旋or不足が歩幅非対称に寄与）

5) 骨盤：前後傾、挙上下制、回旋（可能な範囲で）
- 前後傾の直接数値はなしだが、歩行速度極低下と短歩幅から骨盤の前傾/後傾可動域利用が制限されている可能性
- 挙上下制（上下動）：肩差の平均・標準偏差（shoulder_diff_mean 4.764156 / std 4.029963）とpelvis_swayの大きさから左右の上下動/側方挙下の非対称が示唆される
- 回旋：pelvis_sway 107.592894 と肩差のばらつきから、骨盤回旋の制御不良または代償的回旋が存在する可能性

6) 機能的示唆：どの局面で出やすいか（一般論で可）
- 左歩幅短縮（step_length_l 3.47 cm）は遊脚前進不足あるいは対側（右）立脚期の蹴り出し不足で顕在化しやすく、遊脚期終盤〜初期接地で問題が出やすい
- 低速（speed_m_s 0.044）は全体的に推進期（立脚終期の底屈/膝伸展/股関節伸展）が抑制される場面で表出
- 骨盤横揺れ(pelvis_sway 107.59)は立脚初期〜中期の荷重移動時および片脚支持期に増強されやすい
- かかと傾斜の左右差（heel_tilt L > R, std L大）からは初期接地〜荷重応答（loading response）期の不安定性が疑われる
- 足部回内傾向（ankle_pronation ~159°）は立脚中期の縦アーチの変動や推進効率低下に寄与する可能性

7) 測定限界：pxは相対、撮影条件依存、短尺時不安定など
- 多くの指標は2D解析や画像座標(px)ベースであり、絶対値はカメラ距離・解像度・キャリブレーション依存（step_length_px, speed_px_s などは相対値）
- 角度（knee_varus_valgus, ankle_pronation 等）はランドマーク位置誤差に敏感で、数度の誤差が生じうる
- heel_tilt の平均値が小さい（0.0822 / 0.114457）はノイズや姿勢検出のばらつきに影響されやすい
- 短い撮影距離・歩行回数が少ないと歩行サイクルの代表性が低下し、pelvis_sway や step_asymmetry の信頼性が下がる
- 水平面回旋や股関節の具体的な屈曲/回旋角は2D解析では推定困難であり、回転成分は過小評価または誤推定される可能性
- 臨床応用時は実測値を臨床評価（観察、可動域、筋力、疼痛）と照合して解釈することを推奨

（要旨：右歩幅が左より大きく、歩行速度は非常に遅く、骨盤横揺れと左右非対称が目立つ。左側の遊脚前進または対側の推進不足、立脚期の制御問題が主な懸念。測定はカメラ条件や短尺データに依存するため臨床照合が必要。）

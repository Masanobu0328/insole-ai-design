1) 主要所見（要点・左右差・大振幅）
- 右歩幅が大きく、右 step_length_r = 7.19 cm に対し左 step_length_l = 3.47 cm と明確な左右差（step_asymmetry = 0.697475）がある。  
- 歩行速度が著しく遅い（speed_m_s = 0.044 m/s、speed_px_s = 22.218 px/s）。  
- 骨盤の左右揺れが大きい（pelvis_sway = 107.592894 px）ため側方不安定が示唆される。  
- 両膝はやや内反/外反方向で左右差は小さい（knee_varus_valgus_r_mean = 176.114199°、l_mean = 176.340588°）。  
- 足部では両側にやや回内/過回内（pronation）傾向（ankle_pronation_r_mean = 159.40486°、l = 158.979043°）。  
- かかとの傾きは左の平均と変動がやや大きい（heel_tilt_l_mean = 0.114457、std = 0.117235、右 mean = 0.0822、std = 0.065794）。  
- 肩差（shoulder_diff_mean = 4.764156）が小さく上体の大きな回旋差は目立たないが変動はある（std = 4.029963）。

2) 前額面（XY）: 股関節 内転/外転、膝 内反/外反、足関節 回内/回外
- 股関節：左歩幅短縮に伴う代償的な股関節内転（左側の相対的内転優位）が示唆される（歩幅差に基づく推測）。  
- 膝：knee_varus_valgus は右176.11°、左176.34°で両側やや内反寄り（もしくは軽度の膝関節内側角減少→軽度外反/内反偏位の示唆）。左右差は小さい。  
- 足関節（前額面）：ankle_pronation が右159.40°、左158.98°で両側とも回内傾向を示唆。左はわずかに回内が強い可能性。  
- 骨盤の大きな左右揺れ（pelvis_sway = 107.59 px）は前額面の体幹・骨盤側方移動や荷重偏りを反映している可能性。

3) 矢状面（XZ）: 股関節 屈曲/伸展、膝 屈曲/伸展、足関節 背屈/底屈
- 全体として歩行速度が極端に低いため、股関節伸展（末期立脚での伸展）不足が推定される（十分な前方推進不足）。  
- 膝は立脚終期の伸展・推進が制限され、歩幅短縮（特に左）に関連する屈曲パターンの変化が考えられる。  
- 足関節は底屈（蹴り出し）力が弱く、非効率的なプッシュオフが示唆される（速度・歩幅低下と一致）。  
- かかとの傾きの左右差（heel_tilt）から荷重初期の前後位変動があり、初期接地～荷重応答での矢状面の不安定さが考えられる。

4) 水平面（YZ, 近似）: 股関節・膝 内旋/外旋（可能な範囲で）
- 与えられた指標に水平面回旋の直接数値はないが、左右で大きな歩幅差と骨盤揺れがあるため代償的な股関節/膝の回旋（内旋・外旋の偏り）が生じている可能性が高い。  
- 特に短い左歩幅側ではスイング相の代償回旋（外旋や内旋）が出やすく、足部回内（ankle_pronation）とも併せて過剰内旋寄りのパターンが考えられる（近似的所見）。

5) 骨盤：前後傾、挙上下制、回旋（可能な範囲で）
- pelvis_sway = 107.59 px と大きく、骨盤の側方移動・制御不良が目立つ。  
- 左右の歩幅差に伴い片側で骨盤の挙上/低下（骨盤傾斜：pelvic hike/drop）や回旋代償が生じている可能性がある。  
- 前後傾（前傾/後傾）は直接測定値がないが、歩行速度・推進力低下から前傾位の減少（または体幹安定のための過剰後傾）を示唆することがある。  
- 肩差は小さいが骨盤と肩の協調（肩差 std = 4.03）は変動があり、上半身での代償的動揺もあると考えられる。

6) 機能的示唆：どの局面で出やすいか（一般論で可）
- 初期接地～荷重応答：足部回内（ankle_pronation）とかかと傾き変動（heel_tilt）が影響しやすく、衝撃吸収や膝・股関節の内外反トルクに影響。  
- 立脚中期～終期（支持期の体重移行）：骨盤の大きな側方揺れ（pelvis_sway）が顕著になり、支持位安定性低下で左右の荷重移動が不均等。  
- 末期立脚（蹴り出し）：歩幅縮小（特に左）と歩行速度低下からプッシュオフ不足が生じ、足関節底屈・股関節伸展が不十分。  
- スイング相：左短歩幅により左のスイング開始／終結が早くなり、回旋・屈曲パターンの代償や地面クリアランスへの影響が出やすい。  
- 全般：歩速が非常に遅いため持久性・推進力の低下や恐怖回避的な短縮歩幅パターンの可能性がある。

7) 測定限界：pxは相対、撮影条件依存、短尺時不安定など
- すべての px 値（step_length_r_px = 35.958、step_length_l_px = 17.363、pelvis_sway = 107.592894 等）はカメラ距離・解像度・キャリブレーションに依存する相対値であり絶対長さとの換算は条件依存。  
- 角度値（knee_varus_valgus, ankle_pronation 等）は2D解析やランドマーク推定に基づく推定値で、真の関節中心・三次元回転を完全には反映しない。  
- heel_tilt の単位と基準（0 が何を示すか）に依存するため絶対的評価は注意を要する（heel_tilt_r_mean = 0.0822、l_mean = 0.114457）。  
- 短時間・短尺撮影ではステップ間変動や一過性の代償動作により asymmetry 指標が不安定になる可能性がある（step_asymmetry = 0.697475）。  
- 問診情報がないため既往・痛み・補助具使用の影響を考慮できない。臨床検査（ROM、筋力、感覚、三次元解析）と照合して判断する必要がある。

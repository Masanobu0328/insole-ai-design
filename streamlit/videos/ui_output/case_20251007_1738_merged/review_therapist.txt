1) 主要所見（要点・左右差・大振幅）
- 明らかな歩幅左右差：step_length_r = 7.191512 cm に対し step_length_l = 3.472552 cm（step_asymmetry = 0.697475）で右歩幅が約2倍。
- 非常に低速：speed_m_s = 0.044 m/s（speed_px_s = 22.218 px/s）、歩行全体が著しく遅い。
- 骨盤側方動揺が大きい：pelvis_sway = 107.592894 px。
- 肩の左右差は小〜中等度：shoulder_diff_mean = 4.764156 px（変動あり：std = 4.029963 px）。
- 踵傾斜は左>右：heel_tilt_l_mean = 0.114457 に対し heel_tilt_r_mean = 0.0822（標準偏差は左で大きい：heel_tilt_l_std = 0.117235, heel_tilt_r_std = 0.065794）。
- 膝の静的/動的外反傾向（わずか）：knee_varus_valgus_r_mean = 176.114199°、knee_varus_valgus_l_mean = 176.340588°（双方で180°をやや下回る）。
- 足部回内傾向が両側で同程度：ankle_pronation_r_mean = 159.40486、ankle_pronation_l_mean = 158.979043。

2) 前額面（XY）: 股関節 内転/外転、膝 内反/外反、足関節 回内/回外
- 左歩幅短縮に伴い、左股関節で内転（内側寄り）の負荷増加を疑う（左右差有り、推定）。
- 右は相対的に外転位の補償が出る可能性（右 step_length が大きいため）。
- 膝は両側とも若干の外反（varus）寄り：knee_varus_valgus_r_mean = 176.114199°、knee_varus_valgus_l_mean = 176.340588°（左右差小）。
- 足関節は回内（pronation）傾向が両側で存在：ankle_pronation_r_mean = 159.40486、ankle_pronation_l_mean = 158.979043（左右差は小さい）。
- 踵傾斜の左右差（heel_tilt_l_mean > heel_tilt_r_mean）は左側で足関節の傾き・回内の動的増大を示唆。

3) 矢状面（XZ）: 股関節 屈曲/伸展、膝 屈曲/伸展、足関節 背屈/底屈
- 全体的に矢状面の可動域・推進力が低下していると推定：speed_m_s = 0.044 m/s、両側の歩幅が小さい（特に左）。
- 左側は遊脚位の屈曲（膝屈曲・股関節屈曲）獲得不足により step_length_l = 3.472552 cm と短縮している可能性が高い。
- 右は相対的に歩幅を稼いでおり、右股関節/膝の伸展・推進が残っている（step_length_r = 7.191512 cm）。
- 足関節は底背屈の協調不足（前方推進不足）で、十分な踵離れ（toe-off）・背屈が得られていないことが示唆される（低速・短歩幅より推定）。
- 踵傾斜の標準偏差差（heel_tilt_l_std = 0.117235 > heel_tilt_r_std = 0.065794）は左で矢状面制御の変動が大きいことを示唆。

4) 水平面（YZ, 近似）: 股関節・膝 内旋/外旋（可能な範囲で）
- 水平面データは限定的だが、足部回内（ankle_pronation）が両側であることから、股関節・膝の内旋傾向が二次的に生じている可能性がある。
- 左の踵傾斜増加（heel_tilt_l_mean = 0.114457）と左短歩幅から、左下肢で内旋方向の運動連鎖が出やすい推定。
- ただし明確な回旋角度の直接計測値はないため、水平面の評価は概念的推定に留まる。

5) 骨盤：前後傾、挙上下制、回旋（可能な範囲で）
- 骨盤の側方揺れが大きい：pelvis_sway = 107.592894 px。歩行中の側方安定性の低下を示す。
- 短い歩幅側（左）で骨盤の上下・側方制御が不安定になりやすく、左右不均衡が予想される（短歩幅に関連する骨盤挙下/代償動作の可能性）。
- 骨盤前後傾や回旋の直接値は未提供だが、低速で歩幅非対称が大きいため骨盤回旋による歩幅補償が生じている可能性が高い。
- 肩差（shoulder_diff_mean = 4.764156 px）があるため、上位体幹の代償性回旋や肩帯の偏位も考慮すべき。

6) 機能的示唆：どの局面で出やすいか（一般論で可）
- 左側の遊脚始動～遊脚中期（initial swing～mid swing）で歩幅短縮・屈曲獲得不足が出やすい。
- 立脚単脚支持期（single limb support）で骨盤側方揺れが大きく、左支持期または右支持期での代償が観察されやすい（左右による）。
- 立脚前期～終期（terminal stance～pre-swing）で足関節の背屈/底屈協調不足により推進力が低下、結果として全体速度が極端に低下している（speed_m_s = 0.044 m/s）。
- 方向転換・段差昇降・加速時に不安定性・つまずきが生じやすい（大きな骨盤swayと歩幅非対称より）。
- 左側の足部/踵挙動の変動（heel_tilt_l_std = 0.117235）があるため、左側で着地時・蹴り出し時の不安定性が増す局面が予想される。

7) 測定限界：pxは相対、撮影条件依存、短尺時不安定など
- px値・角度値はカメラ・キャリブレーション・撮影距離に依存：speed_px_s = 22.218 px/s、pelvis_sway = 107.592894 px 等は相対指標。
- step_length_r_px = 35.958 px、step_length_l_px = 17.363 px はピクセル長での比較は可能だが実世界換算はキャリブレーションによる誤差あり。
- 提供された step_length_cm（7.19 cm / 3.47 cm）は極端に小さく、測定条件（カメラ視角・ズーム・被検者位置など）による過小評価の可能性があるため実際の物理長とは差があり得る。
- 関節角（回旋等）は近似であり、正確な三次元解析・座標系／骨模型による検証が必要。
- サンプル数・歩行試行数未提示、短尺撮影や被写体の衣服・視認性がデータ精度に影響する点に注意。
- EMG・力板・臨床検査（ROM,筋力）と組み合わせて評価することを推奨。
